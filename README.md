# ❤️ APulse Heart Tracker Web

**📌 Description**  
A beautiful, fully functional, mobile-first web replica of the **APulse Heart Tracker** Android app.  
Measure heart rate with your camera, log blood pressure & blood sugar, view real-time trends, history, and get health insights — all stored locally on your device with zero data sharing.

**✨ Features**  
- 📸 Instant camera heart rate measurement (real webcam + realistic simulation)  
- 🩸 Manual logging for Blood Pressure & Blood Sugar  
- 📈 Interactive trend chart (HR over time)  
- 📜 Full measurement history with delete option  
- 🏠 Live dashboard with last readings & health tips  
- ⚡ Quick-measure button in header  
- 🔒 100% private – uses browser localStorage only  
- 📱 Fully responsive & touch-optimized for phones & tablets  

**🏷️ Tags / Keywords**  
`heart-rate-monitor-web`, `pulse-tracker`, `blood-pressure-logger`, `blood-sugar-tracker`, `camera-heart-rate`, `health-trends-chart`, `online-pulse-checker`, `apulse-web`, `heart-health-app`, `web-medical-tracker`, `html5-health-monitor`

**🛠️ How to run**  
1. Download all 4 files into the **same folder**  
2. Open `index.html` in any modern browser (Chrome recommended for camera)  
3. Grant camera permission when prompted and start tracking!
